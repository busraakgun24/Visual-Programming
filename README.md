# Visual-Programming
Python arayüz kütüphaneleri ile geliştirilmiş masaüstü uygulamaları ve GUI tasarımları
Visual Programming: Python & PyQt5 Projects
Bu depo, Python dilini ve PyQt5 kütüphanesini kullanarak geliştirdiğim modern masaüstü uygulamalarını, grafiksel kullanıcı arayüzlerini (GUI) ve olay tabanlı (event-driven) programlama çalışmalarımı içermektedir.
Teknik Odak Noktaları
PyQt5 Framework: Qt Designer ve kod tabanlı arayüz geliştirme teknikleri.
Signals & Slots: Kullanıcı etkileşimlerini (buton tıklamaları, girişler) arka plan mantığına bağlayan dinamik yapı.
Layout Management: Farklı pencere boyutlarına uyum sağlayan (responsive) esnek arayüz tasarımları (HBoxLayout, VBoxLayout, GridLayout).
Styling with QSS: Uygulamalara modern bir görünüm kazandırmak için CSS benzeri stil sayfaları (Qt Style Sheets).
Kurulum ve Çalıştırma
Projeleri yerel makinenizde çalıştırmak için gerekli kütüphaneleri şu komutla yükleyebilirsiniz:
Bash
pip install PyQt5
Uygulamayı başlatmak için ana dosyayı (örn: main.py) VS Code terminalinde çalıştırın:
Bash
python main.py
